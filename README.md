# 🎲 Decision Maker App

An interactive tool built in Godot to solve day-to-day choice paralysis. 

## 🚀 Planned Features
- **Phase 1:** Core random-choice generator from user-inputted lists.
- **Phase 2:** Visual excitement (spinning wheel)
- **Phase 3:** Smart suggestions using location/restaurant APIs and Gemini AI context.

## 🛠️ Tech Stack
- **Engine:** Godot 4.x
- **Language:** GDScript (Python-based)
- **Version Control:** Git & GitHub
